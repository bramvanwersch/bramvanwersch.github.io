Roguelike


Description:

This is a project that I worked on for a couple of months. It is a roguelike inspired by one of my favorite games; borderlands. While the game is not finished all the weapons that you can find are randomly generated consisting of multiple wepon parts. Currently the game has one boss one stage and a couple of different enemies. I want to continue development of this game in the future.

Stats

- Language: Python 3
- Main library: Pygame 2.0
- State: Under construction
- Start date: 29 Jan 2020
- Wanted features: more stages, more bosses, more weapons
