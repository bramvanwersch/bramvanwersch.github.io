Machine Mining


Description:

This is my largest project that I have madeso far. It is a game where you command a mining robot to gather resources to build machines to gather more resources. It is very much under construction and is mainly a huge learning process for me. From building a massive application and keeping the overview to designing my own interfaces using pygame, map generation and saving and loading systems. This project was a lot of fun to make but has become hard to maintain due to the lack of tests and constant bugs that appear seaminlgy out of nowhere. The biggest issue however is the engine, were pygame and CPU accelerated graphics become limited even for a simple 2D game like this because of the number of things that need redrawing. Tldr: likely not ever going to be finished but always cherished.

Stats:

- Language: Python 3
- Main library: Pygame 2.0
- State: Under construction and likely never to be finished
- Start date: 12 June 2020
- Wanted features: An end goal, polishing, programmable machines and a lot of bugfixing
