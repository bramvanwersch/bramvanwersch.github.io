Evolution Simulator


Description:

This is my first larger programming project that I have tracked with GitHub. It aims to simulate evolution of 3 groups of species, herbivores, omnivores and carnivores. There are a number of traits that these species have that can evolve. These traits are backed up by actual genomes and DNA and change in a (somewhat) similar fassion as reality. For this implementation however there are so called 'optimal' genes that help define how well the current version of the gene of a species functions compared to the maximal value. This also means that there is no True evolution since there is a set standard that thing evolve towards as opposed to the environment determining the most favorable gene.
